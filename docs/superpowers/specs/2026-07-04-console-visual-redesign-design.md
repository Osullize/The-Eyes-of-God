# 控制台视觉重设计方案

## 目标

将当前 Vue 控制台从普通表单后台提升为更清晰、更有识别度的热泵商机作战台。改动只覆盖前端视觉与局部结构语义，不改变后端 API、数据库、任务逻辑。

## 设计方向

- 左侧导航改为深色命令轨，承载模块切换和产品识别。
- 主工作区保持浅色、高密度、低干扰，适合长时间查看任务、表格和客户画像。
- 顶部模块标题作为当前工作上下文，不做营销式 hero。
- 指标区使用无空隙 bento 栅格，强化数据总览。
- 任务进度、候选组、原始数据表和 AI 画像阅读区使用统一的控制台材质、阴影和动效。
- 字体栈改为 Geist / Cabinet Grotesk 优先，移除 Inter 主导感。

## 约束

- 不引入新 npm 依赖。项目当前没有 GSAP，动效用 CSS transition / transform 实现。
- 不做卡片套卡片。
- 不增加说明型营销文案。
- 不改变任何接口字段和业务流程。

## 验证

- 添加前端视觉静态测试，检查字体、命令轨、bento、表格、AI 阅读页等关键样式。
- 运行相关 Python 测试。
- 运行 `npm run build` 验证 Vue/TypeScript 构建。
